Featured game

Sortiago

Sortiago is a relaxing color sort puzzle game for mobile devices. Players organize colorful pieces into clean, satisfying sets across calm puzzle sessions.

The current app release includes optional one-time purchases through Google Play Billing and uses a consent flow based on Google User Messaging Platform. The current release does not serve ads yet; future versions may include Google AdMob rewarded or interstitial ads after consent where required.
